# Paysim Financial Transaction Analysis Dashboard

This repository contains a Streamlit application for analyzing the Paysim financial transactions dataset. It performs data cleaning, feature selection, K-Means clustering, PCA visualization, and displays various interactive plots.

## Installation

Install dependencies:
pip install -r requirements.txt


## Run

Run the Streamlit app:
streamlit run app.py


## Features

- Transaction type distribution
- Correlation heatmap of features
- K-Means clustering on transaction data
- PCA visualization of clusters
